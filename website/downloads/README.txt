Place the compiled installer here:

  seng-setup-1.0.0-windows-x64.exe

Steps to generate it:
1. Run: prebuild_installer.bat       (from the repo root)
2. Open Inno Setup Compiler
3. File → Open → seng_installer.iss
4. Build → Compile  (F9)
5. Copy installer\seng-setup-1.0.0-windows-x64.exe into this folder
6. Deploy the website\ folder to nocorps.org/seng
