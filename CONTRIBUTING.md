# Contributing to seng

Welcome! We're excited that you're interested in contributing to **seng** (Simple English Programming Language). This project is now public and open to the community to help develop and evolve the language.

## 🚀 How to Contribute

There are many ways to contribute:

1.  **Reporting Bugs**: Found a bug? Open an issue and describe it.
2.  **Suggesting Features**: Have an idea for a new keyword or improvement? Open a feature request.
3.  **Writing Documentation**: Help us make the documentation clearer and more comprehensive.
4.  **Examples**: Add more examples to the `examples/` directory.
5.  **Code Contributions**:
    *   Fork the repository.
    *   Create a new branch for your feature or bugfix.
    *   Submit a Pull Request.

## 🛠 Development Setup

### Prerequisites
*   A C compiler (GCC, Clang, or MSVC).
*   `make` (optional, for Linux/macOS).

### Building
On Windows:
```sh
build.bat
```

On Linux/macOS:
```sh
make
```

### Running Tests
Currently, you can run the interpreter against the files in the `examples/` directory to verify behavior.

## 📜 Code of Conduct
Please be respectful and helpful to others in the community. Our goal is to make programming accessible to everyone.
